# Credit Card UI
A Credit Card UI that can be easily integrated in any eCommerce SPA.


## Live
upcoming!
## Installation

1. Clone the project to your local directory

```
git clone https://github.com/bellom/
```

2. Run `cd credit-card-ui`

3. Run `npm install`

4. Run `npm run build`

5. Run `npm start`
